PP Programação Aplicada I

Atividade 1

Elaborar um trabalho em texto, sem plágio, descrevendo as seguintes etapas que envolvem o desenvolvimento de um software:

  1. Definição do problema
  2. Planejamento da solução
  3. Codificação
  4. Testes
  5. Documentação

Descrever e demonstrar com exemplos o que são e como funcionam fluxogramas e pseudocódigos.

Por fim, elaborar uma conclusão sobre as etapas que envolvem o desenvolvimento de uma solução de software.

Material de apoio:

  1. Introdução ao desenvolvimento de software (material de aula)
  2. Algoritmo e Lógica de Programação - Algoritmos Parte 1 (Apostila)
  3. Algoritmo e Lógica de Programação - Algoritmos Parte 2 (Apostila)
  4. Lógica de Programação Aula 05: Fluxograma ou Diagrama de Blocos (Youtube)
  5. Algoritmo Estruturado - FIAA (Apostila)
